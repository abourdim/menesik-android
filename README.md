# 🕌 Menesik — Android wrapper

Capacitor wrapper that packages the [menesik](https://github.com/abourdim/menesik) web app as an installable **Android APK**.

> مناسك · Menesik

Complete Hajj & Umrah pilgrim guide in three languages (AR/FR/EN). Includes step-by-step rituals, an interactive checklist with progress tracking, and a curated dua collection. Fully offline-capable — essential when there is no signal in Mecca.

## Quick start

```bash
./manage.sh    # auto-detects your OS, then opens the menu
```

Build, test on emulator, and prepare a Play Store release from a single guided menu. See [WORKSHOP.html](WORKSHOP.html) for the full step-by-step guide.

## Download the latest APK

[📦 v1.0-debug release](https://github.com/abourdim/menesik-android/releases/latest) · [![QR](releases/download-qr.svg)](https://github.com/abourdim/menesik-android/releases/latest)

Scan the QR with an Android phone, or `adb install -r menesik-v1.0-debug.apk`.

## Capabilities required at runtime

- Standard WebView (no special hardware)
